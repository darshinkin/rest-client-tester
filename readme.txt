Тестовое приложение для тестирования Друга.

Имеется следующие профиля:
- local - для запуска локально
- dev - для запуска тестирования на dev стенде

запускатеся с помощью мавен команды из корневой директории проекта серверной части:
mvn clean test -Dspring.profiles.active=dev -f rest-client-tester/pom.xml
где -Dspring.profiles.active=dev указывает на запуск с профилем dev
